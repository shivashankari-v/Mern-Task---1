# MERN Stack Internship - Task 1

## 📌 Project Overview

This project is a basic MERN Stack Product Management application developed as part of my internship task. It demonstrates the integration of React, Express.js, MongoDB, and Mongoose to build a full-stack web application.

---

## 🚀 Features

- Product Card Component using React
- Reusable Components with Props
- Express.js Backend Server
- MongoDB Database Integration
- Mongoose Product Model
- REST API for Products
- Fetch API Integration
- Dynamic Product Display

---

## 🛠️ Technologies Used

- React.js

- Node.js

- MongoDB Compass

## 📡 API Endpoint

### Get All Products

```
GET /api/products
```

Returns all products stored in MongoDB.

---

## ▶️ How to Run the Project

### Frontend

```bash
cd frontend
npm install
npm run dev
```

### Backend

```bash
cd backend
npm install
node server.js
```
